# EEG Quantum Net v0.0.3 pre-Alpha

Sistema experimental para procesamiento de EEG + extracción de características + incrustación cuántica (100 qubits) + integración ML.

Este proyecto incluye:
- Procesamiento de señales EEG
- Transformación a features
- Embeddings cuánticos
- Circuitos cuánticos Qiskit
- API en Python
- Interfaz web con React + Vite

It should be noted that this is not entirely the final version of the project; it is only version 0.0.3 of the qmicns system that I had previously designed.
